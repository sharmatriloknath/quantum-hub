# Quantum Hardware

Quantum computers use qubits implemented with various technologies.

## Superconducting Qubits

- Used by IBM and Google
- Operate at millikelvin temperatures

## Example Circuit

```qasm
OPENQASM 2.0;
qreg q[2];
h q[0];
cx q[0],q[1];
```

Learn more at [IBM Q Experience](https://quantum-computing.ibm.com/).

|\Phi^+\rangle = \frac{1}{\sqrt{2}} (|00\rangle + |11\rangle)
